PitchModel

An MLB decision-support tool that uses historical data to forecast the next pitch in real-time.

Launch Application

About the Project

In high-stakes baseball, information is everything. PitchModel was developed to explore the patterns of MLB pitchers and provide fans or coaches with a "most likely" outcome for the next delivery based on the current game state.

By analyzing thousands of data points including pitch count, runner placement, and previous pitch results, the model identifies tendencies that aren't always visible to the naked eye.

How It Works

  • Data Processing: Utilizes R and Statcast data to build a robust foundation of pitch sequences.
  • Logic Engine: Employs decision-tree principles to calculate probabilities based on situational variables like "Innings," "Outs," and "Score Differential."
  • Interface: Built with R Shiny to provide a reactive, user-friendly experience that updates predictions instantly as the user toggles game parameters.
